🚀 Práticas de Manipulação do DOM com JavaScript

Este repositório contém 3 de exercícios práticos desenvolvidos para consolidar conhecimentos sobre o DOM (Document Object Model) e a interatividade em páginas web, seguindo a trilha de aprendizado da FIAP.  

💻 O que foi desenvolvido:

Neste projeto, apliquei conceitos fundamentais de JavaScript para manipular elementos HTML de forma dinâmica, focando em três cenários principais:


Criação de funções para alterar propriedades de CSS (como color, fontSize e backgroundColor) em tempo real através do objeto style.  

Uso do método document.getElementById para capturar elementos específicos e manipular seu conteúdo com innerHTML.  

Implementação de captura de dados de entrada (input) utilizando a propriedade .value.  

Uso do evento onsubmit combinado com event.preventDefault() para processar informações sem recarregar a página.  

Validação visual de campos em tempo real utilizando o evento onchange.  

Desenvolvimento de uma galeria onde miniaturas controlam um visualizador principal.  

Uso estratégico da palavra-chave this para passar referências de elementos como argumentos de funções.  

Aplicação do método setAttribute para modificar dinamicamente o atributo src de imagens.
